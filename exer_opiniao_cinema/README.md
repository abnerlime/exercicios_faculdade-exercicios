Enunciado da Questão: 

Um cinema fez uma pesquisa de satisfação com seus clientes. Para cada cliente, são lidos: 


* Idade  
* Opinião sobre o filme:  
1. Péssimo  
2. Ruim  
3. Regular  
4. Bom  
5. Ótimo  

* A pesquisa termina quando for digitada uma idade negativa.  
* O programa deve:  
1 - Contar quantas pessoas responderam em cada tipo de opinião (1 a 5).  
2 - Calcular a média de idade dos que responderam “Ótimo” (5).  
3 - Encontrar a maior idade entre os que responderam “Péssimo” (1).  
4 - Calcular o percentual de pessoas que responderam “Bom” ou “Ótimo” (4 ou 5) em relação ao total.  


Usar:  

* Estrutura de reprodução (while ou do-while).  
* switch-case para tratar a opinião.  
* Contadores e acumuladores (total de pessoas, total por opinião, soma das idades dos que responderam 5).  
* if-else para atualizar maior idade, validar dados, etc.
